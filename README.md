### Chrome 健康提醒插件

项目结构如下：
```
timer/
├── manifest.json          # 插件配置
├── background.js          # 后台定时与通知逻辑
├── icons/                 # 插件图标
└── popup/
    ├── popup.html         # 设置界面
    ├── popup.css
    └── popup.js
```

### 功能
- 喝水提醒：默认每 45 分钟提醒一次（可调 15–120 分钟）
- 上厕所提醒：默认每 90 分钟提醒一次（可调 30–180 分钟）
- 免打扰时段：可设置夜间不弹通知（如 22:00–08:00），计时仍会继续
- 系统通知：到时间会弹出 Chrome 桌面通知
- 角标倒计时：工具栏图标显示距离下次提醒的剩余时间
- 立即提醒：可手动触发测试通知

### 安装方法
- git clone git@github.com:littleflower212/timer.git
- 打开 Chrome，地址栏输入 chrome://extensions/
- 右上角开启 开发者模式
- 点击 加载已解压的扩展程序
- 选择文件夹：/下载的扩展程序文件夹位置
- 首次安装时，Chrome 会询问是否允许通知，请点击 允许

### 使用说明
点击浏览器工具栏上的插件图标，即可打开设置面板：

- 用开关单独开启/关闭喝水、上厕所提醒
- 拖动滑块调整提醒间隔
- 面板底部会显示距离下次提醒的倒计时
